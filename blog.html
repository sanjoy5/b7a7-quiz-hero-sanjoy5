Qus: What is the difference between map, filter, find, foreach ?

Answer: map(): The map method is used to iterate over each element of an array and calls the provided function for each element. This method creates a new array and doesnâ€™t change the original array.
filter(): The filter method used to find the items which fulfill the given condition. Filter method also creates new array. The filter method does not change the original array.
find(): The find method returns the value of the first element that passes a test. The find method returns undefined if no elements are found. The find method does not change the original array.
forEach(): The forEach method is used to iterate over an array and perform a function on each item. This method does not change the array and it returns undefined.

Qus: What is the difference between Local Storage, Session Storage, and Cookies ?

Answer: Local Storage: Useful for storing data. The stored data is not deleted when the browser is closed. So that the user will access this data later.
Session Storage: Useful for storing data. The data is stored only for the session and will be deleted when the browser is closed.
Cookies: Useful for storing data. The data can be set to expire at a certain time.

Qus: What is event loop in JavaScript ?

Answer: Event Loop: JavaScript executes all operations on a single thread. For the event loop JavaScript can execute asynchronous code. Because javascript has event queue. So the event loop is the secret behind JavaScript's asynchronous programming.
